# cli-menu-query

**标签**: `ready-for-agent`  
**依赖**: graph-data-structure

## 目标

实现命令行菜单，让用户能选择起点和终点并触发最短路径查询。

## 任务内容

1. 在 `src/main.c` 中实现主菜单循环：
   - 显示地点列表（带编号）。
   - 提示输入起点编号。
   - 提示输入终点编号。
   - 调用 Dijkstra 算法并输出结果。
   - 支持返回菜单或退出。
2. 输出格式：`教学楼 → 图书馆`，总距离 `xxx 米`。
3. 界面友好，有分隔线和提示信息。

## 验收标准

- [ ] 用户能输入两个地点编号并看到最短路径结果。
- [ ] 输出包含路径文字序列和总距离。
- [ ] 菜单支持多次查询和退出。

## Review 要求

完成后提交代码，人工 review：
- 界面是否清晰易用
- 输出格式是否符合任务书要求
- 是否有明显的交互流程问题
