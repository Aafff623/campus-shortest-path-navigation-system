# input-validation

**标签**: `ready-for-agent`  
**依赖**: cli-menu-query

## 目标

处理各类异常输入，保证程序不会因错误输入而崩溃。

## 任务内容

1. 对起点/终点编号进行校验：
   - 非数字输入：提示重新输入。
   - 编号越界：提示“地点不存在”。
   - 起点 == 终点：提示“起点和终点不能相同”。
2. 对 Dijkstra 返回的不可达情况给出友好提示。
3. 在 `src/main.c` 中统一处理输入缓冲区，避免 scanf 残留换行符导致的问题。

## 验收标准

- [ ] 输入字母或其他非数字字符时，程序提示重新输入。
- [ ] 输入不存在编号时，程序提示“地点不存在”。
- [ ] 起点和终点相同时，程序阻止查询并提示。
- [ ] 两点不可达时，程序提示“没有可达路径”。

## Review 要求

完成后提交代码，人工 review：
- 各种异常场景是否都被覆盖
- 提示信息是否清晰中文
- 程序在异常输入后能否继续运行
