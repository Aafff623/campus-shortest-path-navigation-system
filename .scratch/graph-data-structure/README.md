# graph-data-structure

**标签**: `ready-for-agent`  
**依赖**: setup-c-project

## 目标

定义图的数据结构，并实现示例地点与路径的初始化加载。

## 任务内容

1. 在 `src/graph.h` / `src/graph.c` 中定义：
   - `Place` 结构体（编号、名称、类型等）。
   - `Graph` 结构体（顶点数、边/邻接矩阵或邻接表）。
2. 实现初始化函数 `graph_init`。
3. 实现添加边函数 `graph_add_edge`（双向，带权）。
4. 内置示例数据：至少 9 个地点、15 条双向路径，与前端原型 `js/app.js` 中的数据一致。
5. 实现打印地点列表和路径列表的函数，供命令行菜单调用。

## 验收标准

- [ ] 程序启动后能正确打印 9 个地点和 15 条路径。
- [ ] 图结构支持通过编号索引地点。
- [ ] 边信息包含起点、终点、长度，且双向存储。

## Review 要求

完成后提交代码，人工 review：
- 结构体设计是否合理
- 示例数据是否与前端原型一致
- 代码是否简洁、注释是否清晰
