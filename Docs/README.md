# Docs 目录说明

本项目文档按用途分目录存放，保持资料可追溯、可交接。

| 目录 | 用途 |
|------|------|
| `backup/` | 手动或自动化备份资料，如数据库快照、代码打包、中间产物。 |
| `原始资料/` | 老师下发的原始任务书、需求文档、参考文件，**只读归档**。 |
| `handoff/` | 交接文档：项目说明、运行方式、目录结构、后续可改进点。 |
| `reports/` | 课程设计说明书、测试报告、截图、验收材料。 |

## 当前已归档

- `原始资料/课程设计任务书.doc` — 老师下发的 Word 版任务书原件
- `原始资料/课程设计任务书.txt` — 任务书文本副本（便于 diff/搜索）
- `../assets/prototype/` — 前端静态原型（来自 `campus-nav-prototype.zip`）

## 命名约定

- 日期类文件：`YYYYMMDD-说明.ext`
- 版本类文件：`文件名-v1.ext`
- 报告类文件：`报告名-姓名-学号.docx`（按学校要求调整）
