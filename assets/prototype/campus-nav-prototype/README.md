# 校园最短路径导航系统 - 前端静态原型

这是一个基于课程设计任务书制作的前端效果原型，适合用于大一/大二小学期项目展示、截图、说明书配图和后续真实开发参考。

## 使用方式

1. 解压压缩包。
2. 双击打开 `index.html`。
3. 也可以用 VS Code Live Server 打开整个文件夹。

## 页面清单

- `index.html`：首页 / 系统入口
- `query.html`：路径查询页
- `result.html`：查询结果页，支持 URL 参数，例如：`result.html?start=teaching&end=library`
- `locations.html`：地点管理页
- `paths.html`：路径管理页
- `algorithm.html`：Dijkstra 算法说明页
- `about.html`：关于系统 / 功能说明页

## 已覆盖功能

- 校园主要地点展示
- 地点管理界面
- 路径管理界面
- 起点 / 终点选择
- Dijkstra 最短路径查询
- 最短距离展示
- 路径文字输出：如 `教学楼 → 图书馆`
- 地图路径高亮
- 未选择地点、起终点相同等异常输入提示
- 蓝白 / 浅绿 / 深色三套主题切换
- 响应式布局，支持较小屏幕预览

## 技术说明

这是纯静态前端原型，不依赖后端、不依赖 npm 包。

- HTML：页面结构
- CSS：视觉样式与响应式布局
- JavaScript：mock 数据、Dijkstra 算法、主题切换、地图渲染

## 后续可改造方向

- 把 `js/app.js` 中的 mock 地点和路径数据换成后端接口数据
- 增加真实地图图片或校园平面图底图
- 将新增 / 编辑 / 删除从静态提示改成真实数据操作
- 用 C / Java / Python 实现后端算法，再由前端调用
