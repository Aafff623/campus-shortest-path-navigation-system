实现思路

地点作为顶点，路径作为边，路径长度作为权值。查询时从起点出发，不断选择当前距离最小的未访问顶点，并更新相邻顶点距离。

1

初始化

起点距离设为 0，其余地点距离设为无穷大，并准备 previous 数组记录前驱节点。

2

选择最小点

在未访问地点中选择当前已知距离最小的地点作为当前节点。

3

松弛边

遍历当前节点的所有相邻路径，若经过当前节点可得到更短距离，则更新距离和前驱。

4

输出路径

到达终点后，从终点沿 previous 数组回溯，得到"起点 → 中间点 → 终点"的最短路径。

function Dijkstra(graph, start, end):
    dist[start] = 0
    dist[other] = INF
    previous[all] = null

    while exists unvisited node:
        current = unvisited node with min dist
        if current == end: break
        mark current as visited

        for each neighbor of current:
            newDistance = dist[current] + edgeWeight
            if newDistance < dist[neighbor]:
                dist[neighbor] = newDistance
                previous[neighbor] = current

    return rebuildPath(previous, end), dist[end]
图结构 顶点保存地点，边保存两个地点之间的距离。
复杂度 基础实现约 O(V² + E)，适合课程设计规模。
限制 Dijkstra 要求路径权值不能为负数。
