使用说明

  1. 把真实底图（官方 JPG / 3D 投影 PNG）替换 images/ 下的文件。
  2. 修改 campus-layout.json 中 map.image 指向新图。
  3. 点击每个地点中心，记录 x、y，更新 JSON 中对应 place 的坐标。
  4. 沿道路点击拐点，组合成 edgePolylines 数组。
  5. 保存后刷新本页验证。
