# Triage labels

These labels are used by the `triage`, `to-issues`, and related skills.

| Label | Meaning |
|-------|---------|
| `needs-triage` | Maintainer needs to evaluate |
| `needs-info` | Waiting on reporter / more context needed |
| `ready-for-agent` | Fully specified; an AFK agent can pick it up |
| `ready-for-human` | Needs human implementation or decision |
| `wontfix` | Will not be actioned |

Use these exact strings when marking issues.
