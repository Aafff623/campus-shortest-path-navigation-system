# 运行说明

## 环境要求

- Windows 10/11 或兼容 POSIX 的环境
- gcc 或 MinGW-w64（`gcc --version` 能正常输出）
- make（Windows 下可用 MinGW 自带的 `mingw32-make`，或安装 GNU Make）

## 目录结构准备

```bash
assets/data/   # 用于存放 C 程序导出的 routes.json
bin/           # 编译产物
src/           # C 源码
```

## 编译

```bash
make build
```

或手动：

```bash
gcc src/*.c -o bin/campus_nav.exe
```

## 运行交互模式

```bash
make run
```

或手动：

```bash
./bin/campus_nav.exe
```

## 导出 JSON 供前端使用

```bash
make export
```

或手动：

```bash
./bin/campus_nav.exe --export
```

导出成功后，`assets/data/routes.json` 会被更新。

## 运行前端页面

前端为纯静态页面，可直接用浏览器打开 `assets/prototype/campus-nav-prototype/index.html`。

推荐使用 VS Code 的 Live Server 插件，或 Python 简单 HTTP 服务器：

```bash
cd assets/prototype/campus-nav-prototype
python -m http.server 8080
```

然后在浏览器访问 `http://localhost:8080`。
